# LDH's Web Resume v2
이전에 HTML, CSS를 기본으로 CDN을 이용해 간단한 반복 요소 Vue.js로 처리했던 것을 Vue CLI로 바꾸는 작업을 진행합니다.


## 개발 환경
- OS: Ubuntu 18.04
- node: 12.8.3
- npm: 6.4.1
- Vue CLI: 4.5.12
